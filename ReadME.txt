To run this project :
 - first clone the repository with the command: git clone https://github.com/emmanuel-montblanc/Software-Development-Skills-Mobile.git
 - Then open android studio, go to 'File', then 'Open' and select the 'TicTacToe' folder inside the repository you've just cloned.
 - Once the project has loaded, you can execute the project by clicking the green arrow or by pressing Shift+F10

Once you cloned the repository, you can also create an APK to get the app on your android device, by clicking in android studio on 'Build' then 'Build bundle(s)/APK(s)', 'Build APK(s)'. You will then get the apk file on your computer and can install it on your phone by downloading this file on your phone and clicking on it to exectute it.
